New Features:
- Auto Mixology Plugin: Automatically handles potion mixing to save you time.

Fixes and Updates:
- UI Stability: Fixed crashes and bugs to improve the user interface..
- Pathfinding Fixes: Better movement around agility shortcuts and tricky areas..
- Combat and Antiban Updates: More reliable antiban features and improved combat logic..

Performance and Code:
- Faster Performance: Scripts are now optimized for quicker actions and smoother gameplay..
- Cleaner Code: We’ve cleaned up and simplified the code, making it easier to update and maintain..

New Features

Tormented Demons plugin

Full Automation

Handles banking, traveling, combat, and retreating without manual input, so you can focus on your strategy.

Banking & Restocking

Uses a predefined inventory setup called "tormented" to withdraw and restock supplies, including a dueling ring for teleportation.

Traveling & Combat

Automatically travels to Tormented Demons using a Guthixian Temple teleport and engages in combat efficiently.

Auto-Retreat

When supplies run low, the bot retreats to the Ferox Enclave using a dueling ring, restocks, and resumes the task.

Combat-Only Mode

Jump straight into combat without restocking or traveling. Ideal for situations where you’re already near the demons.

Setup Made Easy

Place your character near the Tormented Demons, and the bot takes care of combat, health, and prayer management. (Note: This mode does not include retreating.)

Auto Prayer & Gear Switching

Adjusts prayers and gear automatically based on the demon’s attack style for improved efficiency.

Custom Loot Management

Configure which items to loot by specifying their names, keeping your inventory clean and efficient.

How to Start
  • Configure Your Settings: Choose between Full Auto or Combat-Only modes, set loot preferences, and adjust prayer settings as needed.
  • Start the Plugin:
    • Full Auto Mode: Make sure the "tormented" inventory setup is ready with supplies, including a dueling ring, and let the bot handle the rest.
    • Combat-Only Mode: Position your character near the demons, and the bot will manage the combat for you.
Aerial Fishing Plugin

Automatically converts any herbs, grimy herbs, or unfinished potions in the inventory into paste. Allows customization of the amount of paste to create in one batch, continuously refining paste as needed. Switches to the mixing room to fulfill potion orders. Efficiently mixes multiple potions simultaneously based on your priority settings: Experience Priority: Focuses on mixing orders within the best available tiers your account can handle, maximizing XP gains or selecting the best available order. Points Priority (Mox/Aga/Lye/Balanced): Prioritizes mixing orders that provide maximum priority points (20) or selects the best available order. Includes a "Quick Action" option to rapidly create and fulfill multiple potion orders at once.

Mixology Plugin

Mixology Plugin: A new fully automated fishing plugin that does aerial fishing at Molch Island.

Varrock Museum Cleaner

Cleans finds at the museum, drops everything but the lamps :)

You must have the Gloves and Boots equipped, with the Trowel, brush and pick in your inventory. Start in the room with the finds.

Bird nest support for woodcutting plugin

Added a toggle for bird nest looting + banking

You must have the Gloves and Boots equipped, with the Trowel, brush and pick in your inventory. Start in the room with the finds.

Miscellaneous
Mahogany Homes Plugin

The plugin has been made more efficient and user-friendly with improvements to pathing logic, object handling, and door handling.

New feature: Users can now toggle the display of log messages in chat, allowing for better monitoring of the plugin's activity.

Tempoross Plugin

A new overlay has been added that helps users monitor their progress in the Tempoross minigame, including a progress bar and real-time updates on the current state and actions.

Woodcutting Plugin

The plugin now functions more efficiently, preventing it from getting stuck on depleted redwood trees by updating the object finder method.

Auto Mining Plugin

The plugin now properly handles banking interactions in Shilo Village, improving efficiency for gem mining activities.

A new configuration option has been added for better continuity with other plugins.

Cannonball Smelter Plugin

Support for the double ammo mould has been added, allowing users to produce cannonballs at double the rate, significantly boosting productivity.

AIO Fighter Plugin

The method for selecting attackable NPCs has been improved, ensuring better targeting efficiency and combat performance.

QoL Plugin
  • New Fletching Section: Added multiple options to enhance the fletching experience:
    • Quick Fletch Items: Allows quick fletching of logs into selected items with one click on the knife.
    • Fletching Item Selection: Choose which item to fletch from logs (e.g., Arrow Shafts).
    • Quick Fletch Darts: Enables rapid fletching of darts. (Note: The make-x setting must be disabled for this to work.)
    • Quick Fletch Arrows: Allows quick assembly of arrows with one click on any type of arrowhead.
    • Quick Fletch Bolts: Enables rapid fletching of bolts. (Note: The make-x setting must be disabled for this to work.)
    • Quick Fletch Headless Arrows: Streamlines the creation of headless arrows with one click on the arrow shafts.
  • Dialogue Section: Added support for handling quest dialogues. The plugin will now automatically handle quest dialogues and select the correct options. (Note: This feature relies on the Quest Helper plugin to highlight the correct dialogue options.)
  • Removed Settings: Removed the Auto Run setting due to the official client's recent update that added the feature.

Added: Deposit chest support for the Gem Mine.
