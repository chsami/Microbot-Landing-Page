Version Updates

  • Bumped versions for various components, including RuneLite (1.10.47-SNAPSHOT, 1.10.46).

Features Added

  • Plugins: New plugins for Arceuus Runecrafting (blood runes support) and Jewelry and Air Orb.
  • Support: Ring of dueling and Pool of Refreshment integration.
  • Enhancements:
    • Natural Mouse: Thread isolation to prevent freezes.
    • Dev Tools: Added Tile Objects selector and widget search functionality.
    • Rs2Player enhancements for potion usage and item checks.
    • Search options for text/name fields and Discord panel updates.

Fixes

  • Improved plugin robustness:
    • Resolved pathfinding, banking, and teleportation issues.
    • Made various scripts more thread-safe and error-resilient.
  • Addressed API deprecations, unnecessary imports, and debug messages.
  • Fixed issues in plugins like Stall Thieving, Fletching, and Motherlode Mine.
  • Antiban validation improvements to prevent out-of-bounds errors.

Improvements

  • Texture limit increased in GPU settings.
  • Pathfinding updates for better wilderness navigation.
  • Refined player combat range checks and banking logic.
  • Timetracking for League Farmticks adjusted to one minute.

Chores and Cleanup

  • Removed unused variables, imports, and redundant methods.
  • Cleaned up banking logic and debug messages.
  • Enhanced Antiban UI with explanations and holiday-themed updates.

Collaboration and Merges

  • Merged multiple pull requests from contributors across various features and fixes.
  • Collaboration with contributors like g-mason0, Hosea-MP, Yt, and SexyPinata.

Special Updates

  • Antiban Duck updated with a holiday theme.
  • Integration of Sunlit Bracers healing buff and gear updates.

Miscellaneous Enhancements

  • DynamicObject API extension for model buffer handling.
  • Adjustments to the Rs2Walker and added randomization methods.

That's all for version 1.6.5.
Happy botting!

- Mocrosoft
